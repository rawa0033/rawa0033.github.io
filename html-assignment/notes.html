HTML Tags and Their Descriptions

<h1> - <h6>
Defines HTML headings, with <h1> being the highest and <h6> the lowest.
<p>
Defines paragraph.
<article>
Represents a self-contained composition in a document.
<img>
Embeds an image into the document.
<ul> & <li>
Defines an unordered list with list items.
